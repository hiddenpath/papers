# papers

Paper source packages and generated artifacts for the EN paper set.

## Structure

- `EN/paper1_cursor/`: converted LaTeX bundle, generated PDF, and original Markdown source for paper 1
- additional papers can be added as sibling directories under `EN/`

## Workflow

Each paper directory should contain:

- the original Markdown source
- the generated LaTeX project
- the compiled PDF
- notes on normalization and conversion decisions
